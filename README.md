# Nosotros
Una app web con galería de fotos y calendario compartidos.

## Características

- Galería de fotos compartida
- Calendario de eventos
- Login/Registro sin validación de email
- Diseño con Tailwind CSS

## Tecnologías

- Next.js 14
- React 18
- Supabase
- Tailwind CSS
- TypeScript

## Instalación
```bash
npm install
npm run dev
```

Abre http://localhost:3000

## Uso

1. Registrate con email y contraseña
2. Sube fotos en la galería
3. Crea eventos en el calendario
